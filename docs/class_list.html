Classes
	HSB_COLOR
			-- A color describe using the Hue-Saturation-Brightness model.
			-- (This is same as HSL but I like the term "brightness" better
			-- than "lightness".)
			-- The model uses integal values instead of reals, except when
			-- converting from and to EV_COLOR.
			-- The hue, saturation, and brightness are NATURAL_8 values.
			-- For ease of use, though, the setter feature take INTEGER_32 values.

	HSB_COLOR_FACTORY
			-- Can be used as a factory for some HSB_COLOR objects.
			-- These features return a new object when called, so that
			-- the basic colors are unchanged.
			-- 
			-- The names of the colors were taken from a Wipipedia article
			-- at:  https://en.wikipedia.org/wiki/Tertiary_color

	HSB_COLOR_TABLE
			-- Table holding HSB_COLOR paired with a name, allowing
			-- seach by string to find a color or search by color to
			-- find a name.
